# 🏀 NBA Analytics & Sports Betting AI

An intelligent sports betting assistant that leverages multi-agent workflows to analyze NBA player props, historical trends, and real-time market data. Built with **Python**, **CrewAI**, and **Claude 3**.

---

## 🚀 Overview
This tool automates the tedious process of cross-referencing player statistics, injury reports, and betting lines. By utilizing an agentic workflow, the system "thinks" through a bet like a professional analyst—evaluating **Points, Rebounds, and Assists (PRA)** and **Over/Under** lines to find edge in the market.

## 🛠️ Tech Stack
* **Language:** Python 3.10+
* **Orchestration:** [CrewAI](https://www.crewai.com/) (Agentic Workflows)
* **LLM:** [Anthropic Claude 3](https://www.anthropic.com/claude) (High-reasoning intelligence)
* **UI Framework:** [Streamlit](https://streamlit.io/)
* **Search Engine:** [Tavily AI](https://tavily.com/) (Real-time sports data retrieval)
